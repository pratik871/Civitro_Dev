UIDAI Certificate Placeholder
==============================

Download the actual UIDAI certificates from:
  https://uidai.gov.in/ecosystem/authentication-devices-documents/public-key

Required files:
  - uidai_auth_sign.pem   (Code signing certificate)
  - uidai_root_ca.pem     (Root CA certificate)

For development/testing, the "offline-dev" provider mode skips
certificate chain validation.
